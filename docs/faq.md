## 一般的な質問

### Q1: 「しのうたタイム」とは何ですか？

**A:** 「しのうたタイム」は、VTuber「幽音しの」さんの配信で歌唱された楽曲を検索・閲覧できる非公式ファンサイトです。曲名やアーティスト名で検索し、YouTubeの該当箇所に直接ジャンプできます。

### Q2: このサイトは公式ですか？

**A:** いいえ、本サイトは非公式ファンサイトです。幽音しのさんご本人および所属事務所とは一切関係ありません。ファンが有志で作成・運営しています。

### Q3: 利用料金はかかりますか？

**A:** いいえ、完全無料でご利用いただけます。

### Q4: スマートフォンやタブレットでも使えますか？

**A:** はい、レスポンシブデザインを採用しているため、スマートフォン、タブレット、PCなど、さまざまなデバイスで快適にご利用いただけます。

### Q5: 会員登録は必要ですか？

**A:** いいえ、会員登録は不要です。サイトにアクセスするだけですぐにご利用いただけます。

## 検索機能について

### Q6: 検索しても結果が表示されません

**A:** 以下の点を確認してください：

1. **キーワードのスペルを確認**
   - 曲名やアーティスト名のスペルが正しいか確認してください
   - 全角・半角の違いにも注意してください

2. **キーワードを短くする**
   - 部分一致で検索されるため、キーワードを短くしてみてください
   - 例: 「夜に駆ける」→「夜に」または「駆ける」

3. **検索ボタンをクリック**
   - キーワードを入力した後、必ず「検索」ボタンをクリックしてください

4. **ライブ配信タイトル検索を試す**
   - 「検索対象にライブ配信タイトルを含める」チェックボックスをオンにしてみてください

### Q7: 検索は大文字小文字を区別しますか？

**A:** いいえ、検索は大文字小文字を区別しません。「YOASOBI」「yoasobi」「YoAsObI」はすべて同じ結果になります。

### Q8: 複数のキーワードで検索できますか？

**A:** 現在のバージョンでは、単一のキーワードのみ対応しています。複数のキーワードを入力した場合、それらを含む文字列として検索されます。

例: 「夜に 駆ける」と入力すると、「夜に 駆ける」という文字列を含む楽曲が検索されます。

### Q9: ライブ配信タイトルで検索するメリットは何ですか？

**A:** 特定のテーマの配信で歌われた楽曲をまとめて検索できます。

**使用例:**
- 「歌枠」で検索 → 歌枠配信で歌われた楽曲
- 「耐久」で検索 → 耐久配信で歌われた楽曲
- 「記念」で検索 → 記念配信で歌われた楽曲

**注意:** ライブ配信タイトルを検索対象に含めると、検索結果が大幅に増える場合があります。

### Q10: 検索結果の並び順はどうなっていますか？

**A:** 検索結果は以下の順序で並びます：

1. **配信日の降順**（新しい配信が上）
2. **同一日に複数配信がある場合**: LIVE_IDの昇順
3. **同一配信内**: タイムスタンプの昇順（歌唱順）

これにより、最新の配信から順に、配信内での歌唱順に楽曲が表示されます。

### Q11: 検索をクリアして全件表示に戻すには？

**A:** 以下の方法があります：

1. **検索ボックスを空にする**
   - 検索ボックスの内容を削除
   - 「検索」ボタンをクリック

2. **ページを再読み込み**
   - ブラウザの更新ボタンをクリック
   - または F5 キーを押す

## YouTubeリンクについて

### Q12: YouTubeリンクをクリックしても動画が再生されません

**A:** 以下の原因が考えられます：

1. **配信動画が削除または非公開**
   - 配信動画が削除されたり、非公開になっている可能性があります
   - 別の楽曲のリンクを試してみてください

2. **ポップアップブロック**
   - ブラウザのポップアップブロック機能が有効になっている可能性があります
   - ブラウザの設定でポップアップを許可してください

3. **インターネット接続**
   - インターネット接続を確認してください
   - YouTubeにアクセスできるか確認してください

4. **ブラウザの問題**
   - ブラウザのキャッシュをクリアしてみてください
   - 別のブラウザで試してみてください

### Q13: YouTubeリンクは新しいタブで開きますか？

**A:** はい、YouTubeリンクをクリックすると、新しいタブで動画が開きます。元のページはそのまま残ります。

### Q14: YouTubeリンクをクリックすると、曲の途中から再生されます

**A:** これは正常な動作です。「しのうたタイム」のYouTubeリンクは**タイムスタンプ付きURL**です。リンクをクリックすると、配信動画の該当の歌唱箇所から再生が開始されます。

曲の最初から視聴できるため、配信全体を見る必要がありません。

### Q15: 曲目番号の「1-3曲目」とは何ですか？

**A:** 同一日に複数の配信がある場合、「配信番号-曲順」形式で表示されます。

- **「1-3曲目」**: 1番目の配信の3曲目
- **「2-1曲目」**: 2番目の配信の1曲目

同一日に配信が1つのみの場合は、「3曲目」のように曲順のみが表示されます。

## データについて

### Q16: データはどのくらいの頻度で更新されますか？

**A:** データは有志のファンによる手作業で更新されています。更新頻度は一定ではありませんが、できる限り最新の情報を反映するよう努めています。

配信後、数日から数週間程度でデータが追加される場合が多いです。

### Q17: 楽曲情報に誤りを見つけました

**A:** 大変申し訳ございません。データは手作業で入力しているため、誤りがある可能性があります。

誤りを見つけた場合は、About Usページに記載されている連絡先（[@kajuen_kajuen](https://x.com/kajuen_kajuen)）までご連絡ください。ご協力ありがとうございます。

### Q18: 最新の配信の楽曲がまだ追加されていません

**A:** データの更新には時間がかかる場合があります。有志のファンが手作業でデータを入力しているため、最新の配信の情報が反映されるまで数日から数週間程度かかることがあります。

ご理解とご協力をお願いいたします。

### Q19: データの正確性は保証されていますか？

**A:** 申し訳ございませんが、データの正確性は保証できません。本サイトのデータは有志のファンによる手作業での情報収集と入力に基づいています。

一部情報に誤りがある可能性や、最新の情報が反映されていない場合があります。正確な情報は、公式チャンネルや配信アーカイブをご確認ください。

### Q20: 楽曲リストの並び順がおかしいです

**A:** 楽曲リストページは現在β版です。漢字のソート順が完全ではない可能性があります。

アルファベットのアーティスト名は正しくソートされますが、日本語（特に漢字）のアーティスト名のソート順が期待と異なる場合があります。

正式版リリース時に改善される予定です。

### Q21: 同じ楽曲が複数回表示されています

**A:** 以下の理由が考えられます：

1. **複数回歌唱された**
   - 同じ楽曲が異なる配信で複数回歌唱された場合、それぞれの歌唱が個別に表示されます
   - これは正常な動作です

2. **データの重複（β版の制約）**
   - 楽曲リストページ（β版）では、一部の楽曲が重複して表示される場合があります
   - 正式版リリース時に改善される予定です

## 表示・パフォーマンスについて

### Q22: 「さらに25件表示」ボタンが表示されません

**A:** 以下の理由が考えられます：

1. **検索結果が25件以下**
   - 検索結果が25件以下の場合、ボタンは表示されません
   - すべての結果が既に表示されています

2. **全件表示済み**
   - すでに全件が表示されている場合、ボタンは表示されません
   - 代わりに「全ての○○件が表示されています。」というメッセージが表示されます

### Q23: 初期表示が25件なのはなぜですか？

**A:** ページの読み込み速度を維持するためです。

検索結果が数百件、数千件になる場合、すべてを一度に表示するとページの読み込みが遅くなり、ブラウザの動作が重くなります。

段階的表示機能により、以下のメリットがあります：
- 高速な初期表示
- 必要な分だけ表示
- スムーズなスクロール

### Q24: ページの読み込みが遅いです

**A:** 以下を試してください：

1. **ブラウザのキャッシュをクリア**
   - ブラウザの設定からキャッシュをクリアしてください

2. **他のタブやアプリを閉じる**
   - メモリを解放するため、不要なタブやアプリケーションを閉じてください

3. **インターネット接続を確認**
   - インターネット接続が安定しているか確認してください

4. **ブラウザを変更**
   - 別のブラウザで試してみてください
   - 推奨ブラウザ: Chrome、Firefox、Edge、Safari

5. **段階的表示を活用**
   - 初期表示は25件に制限されているため、通常は高速に表示されます
   - 必要な分だけ「さらに25件表示」ボタンで追加してください

### Q25: テーブルの列幅を調整できますか？

**A:** 申し訳ございませんが、現在のバージョンでは列幅の調整機能はありません。

列幅は自動的に調整されますが、アーティスト列のみ改行が許可されており、他の列は改行されません。

## トラブルシューティング

### Q26: エラーメッセージが表示されます

**A:** エラーメッセージの内容に応じて対処してください：

#### 「ファイルが見つかりません」エラー

**原因:** データファイルが存在しないか、パスが間違っています。

**対処法:**
- ページを再読み込みしてください
- 問題が解決しない場合は、サイト管理者にお問い合わせください

#### 「読み込み中にエラー」メッセージ

**原因:** データファイルの読み込みに失敗しました。

**対処法:**
- ページを再読み込みしてください
- ブラウザのキャッシュをクリアしてください
- 問題が解決しない場合は、サイト管理者にお問い合わせください

#### 「style.css が見つかりません」エラー

**原因:** CSSファイルが見つかりません。

**対処法:**
- ページを再読み込みしてください
- スタイルが適用されていなくても、機能は使用できます
- 問題が解決しない場合は、サイト管理者にお問い合わせください

### Q27: 検索結果が正しく表示されません

**A:** 以下を試してください：

1. **ページを再読み込み**
   - ブラウザの更新ボタンをクリック（F5キー）

2. **ブラウザのキャッシュをクリア**
   - ブラウザの設定からキャッシュをクリア

3. **別のブラウザで試す**
   - 別のブラウザで同じ検索を試してみてください

4. **検索キーワードを変更**
   - 別のキーワードで検索してみてください

### Q28: テーブルのスタイルが崩れています

**A:** 以下を試してください：

1. **ページを再読み込み**
   - ブラウザの更新ボタンをクリック（F5キー）

2. **ブラウザのキャッシュをクリア**
   - ブラウザの設定からキャッシュをクリア

3. **ブラウザのズームレベルを確認**
   - ブラウザのズームレベルが100%になっているか確認
   - Ctrl+0（Windows）または Cmd+0（Mac）でリセット

4. **別のブラウザで試す**
   - 別のブラウザで表示を確認してください

### Q29: 情報ページのTwitter埋め込みが表示されません

**A:** 以下の原因が考えられます：

1. **埋め込みコードファイルが存在しない**
   - 「情報: 表示するコンテンツがありません」というメッセージが表示されます
   - これは正常な動作です

2. **Twitterの読み込みに時間がかかる**
   - Twitter埋め込みの読み込みには時間がかかる場合があります
   - しばらく待ってから再読み込みしてください

3. **広告ブロッカーの影響**
   - 広告ブロッカーがTwitter埋め込みをブロックしている可能性があります
   - 広告ブロッカーを一時的に無効にしてみてください

### Q30: YouTube動画が埋め込み表示されません

**A:** 以下を試してください：

1. **ページを再読み込み**
   - ブラウザの更新ボタンをクリック

2. **YouTubeにアクセスできるか確認**
   - 別のタブでYouTubeにアクセスできるか確認してください

3. **広告ブロッカーを無効化**
   - 広告ブロッカーがYouTube埋め込みをブロックしている可能性があります

4. **ブラウザの設定を確認**
   - ブラウザの設定で、埋め込みコンテンツが許可されているか確認してください
